# eAmbrosia – Frequently Asked Questions

The FAQ page explains how to use the eAmbrosia database, which hosts three registers (Wine, Agricultural products & foodstuffs, Spirit drinks). It details search options, export functions, status meanings (Applied, Published, Registered, Rejected, Cancelled), file number structure, and differences in information displayed for various application statuses. The page also covers how to contact support, links to related resources such as Traditional Specialities Guaranteed (TSG) and Traditional Terms for wine, and notes that statistical graphs are not available directly but can be generated from exported Excel data.

Key points:
- Search tools allow filtering by product type, category, name, application status, country, etc.
- Results can be exported to Excel via a small arrow button.
- Statuses indicate the stage of the registration process.
- File numbers encode protection type, country code, registration number and amendment sequence.
- Information shown on detail pages varies with status, domain, and file age.
- Legal documents are downloadable for registered products.
