# AWRI Ferment Simulator User Guide

The document is a 12‑page PDF that explains how to use the **AWRI Ferment Simulator**, an Excel/LibreOffice spreadsheet tool for wine producers. It covers:
- Purpose and background of the simulator.
- Key features: up to 100 concurrent ferments, temperature & nutrient control, refrigeration demand, what‑if analysis.
- Step‑by‑step instructions for enabling macros, entering data on various worksheets (Home, Basic Ferment Entry, Full Ferment Log, Full Vintage Planner, Refrigeration & Weather, What‑If Analysis, Ferment Archive).
- Practical guidance on mixing/agitation values, nutrient additions, and archiving.
- Contact details for Dr. Richard Muhlack.

The guide is intended as a technical manual rather than marketing material.
