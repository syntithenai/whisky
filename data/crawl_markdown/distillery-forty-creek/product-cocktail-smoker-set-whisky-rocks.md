# Cocktail Smoker Set with Whisky Rocks
The Forty Creek Cocktail Smoker Set is a kit that lets home bartenders add smoky aromas to spirits or cocktails. It uses Native American white oak and includes an apothecary smoker, butane torch (butane not supplied), four tins of smoking fuel (apple, cherry, oakwood, walnut), a wood‑chip spoon, cleaning brush, and three whisky rocks. The set is priced at **$79.95**.

## Product Highlights
- Compatible with many glassware types (Glencairn, Rocks, etc.)
- Includes 3 whisky rocks for serving
- Comes in a durable white‑oak smoker
