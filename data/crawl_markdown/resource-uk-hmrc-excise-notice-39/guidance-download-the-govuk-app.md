# GOV.UK App

The GOV.UK app is a free mobile application for iPhone and Android that lets users access UK government services and information more quickly and personally. It allows you to save topics, keep a history of visited pages, find local council links by postcode, receive notifications about deadlines, and browse GOV.UK content.

## Key Details
- **Availability**: iOS 16+ (min 24.8 MB) and Android 10+ (min 8.4 MB)
- **Download Links**: [App Store](https://apps.apple.com/), [Google Play](https://play.google.com/)
- **First Release**: Beta version, regularly updated based on user feedback
- **Related Apps**: GOV.UK One Login, HMRC app, UK Immigration ID Check, etc.
