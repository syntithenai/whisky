# Andie Bulley Wins Inaugural 78° Create a Stir Sunshine State Gin Cocktail Competition

- **Event**: First 78° Sunshine State Gin cocktail contest held at Purple Palm Bar, Brisbane (4 Apr 2023).
- **Judges**: Daniel Gregory (The Gresham), Marco Nunes (Allara Learning), Jared Thibault (FnB Collective).
- **Winner**: Andie Bulley – *Sunny State Punch*.
  - Uses fruit‑waste verjuice and passionfruit mead; showcases Sunshine State Gin’s tropical blend.
  - Prize: $500 voucher + $500 cash.
- **Product**: 78° Sunshine State Gin – dry, 42% ABV, made with mangoes, pineapple, passionfruit from Queensland. Now part of core range and available online, at the cellar door (Lot 100) and select retailers.

More competitions are planned in other states.
