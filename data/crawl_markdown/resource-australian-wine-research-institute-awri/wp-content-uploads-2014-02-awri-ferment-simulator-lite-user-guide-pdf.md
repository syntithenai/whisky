# AWRI Ferment Simulator Lite User Guide

The document explains how to use the AWRI Ferment Simulator – an Excel‑based spreadsheet tool that predicts wine fermentation performance and associated metrics such as refrigeration load, energy cost and throughput. Two versions exist: a **Lite** version (Office 2003+) for basic simulation of up to three ferments, and a **Full** version (Office 2007+) with advanced features like multiple ferments and refrigeration demand. A LibreOffice variant is also available.

Key sections:
1. **Operating Procedure** – enable macros, navigate worksheets.
2. **Worksheets Overview** – Ferment 1‑3 sheets for data entry; Refrigeration & Weather sheet for climate and plant details; What‑if Analysis and Chart tabs.
3. **Getting Started** – how to input baumé/temperature data, update predictions, record operations (pump‑overs, agitation, nutrient additions), set desired ferment duration, interpret status colours.
4. **Refrigeration & Weather** – enter forecasted max/min temperatures, brine temperature and electricity cost; CO P is auto‑estimated.
5. **What‑if Analysis** – copy a ferment log to the What‑if sheet, modify parameters, recalc predictions, view results on the chart.
