# Position Description – Victorian Distillers Program Industry Development Officer (February 2026)

This document outlines the role, responsibilities, selection criteria and application details for an **Industry Development Officer** supporting Victoria’s distilling sector. The position is funded by the Victorian Government, combines program delivery with one day per week support to Spirits Victoria Association (SVA), and runs until 30 June 2027.

## Key Points
- **Role focus:** Sustainable growth, promotion, capability building, export readiness, visitor‑economy value.
- **Program oversight:** Distillery Door Program streams – grants, training, infrastructure, export initiatives; implementation of the Victorian Distilled Spirits Strategy 2025‑2030.
- **Governance & operations:** SVA committee coordination, reporting, grant application support, KPI monitoring.
- **Stakeholder engagement:** Build relationships with distillers, government, regional bodies, national associations; represent industry interests.
- **Selection criteria:** Program delivery experience, project management, communication skills, strategic thinking, stakeholder engagement. Knowledge of regulatory frameworks and export markets is desirable.
- **Contract details:** Full‑time equivalent (flexible hours), fixed term until 30 June 2027, probation 3 months, salary commensurate with experience.
- **Application deadline:** 23 February 2026 – submit to kylie@australiandistillers.org.au.
