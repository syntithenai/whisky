# Lark Distilling launches Christmas Cask II

Lark Distilling Co., Tasmania, has released its second‑year seasonal single‑malt whisky, **Christmas Cask II**. The spirit is sourced from Old Hobart Distillery and finished in Tasmanian red wine casks previously used by Jean‑Pascal Patisserie for fruit mince pies. The result is a “liquid plum pudding” with floral, citrus, spice and subtle peat notes, described as *Christmas in a glass*.

- **Production**: Old Hobart single malt + Tasmanian red wine casks (fruit jus from mince pies). 
- **Tasting notes**: custard‑covered plum pudding, raisins, cinnamon, nutmeg, orange peel; citrus, spice, mulled wine warmth; Tasmanian apples, cloves, cinnamon, wisp of peat smoke. 
- **Release context**: second year partnership with Jean‑Pascal Patisserie.

For more details visit the Lark Distillery website or the press release page.
