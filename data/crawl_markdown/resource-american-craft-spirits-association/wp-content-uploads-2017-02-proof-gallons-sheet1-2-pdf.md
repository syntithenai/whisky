# Proof Gallons

This PDF provides a conversion table that translates the number of proof gallons produced into total liters, total gallons, and the equivalent number of cases (12 × 750 ml bottles = 9 L per case). The table lists values for production volumes ranging from 1 to 750,000 proof gallons.

## Conversion Details
- **Proof Gallons**: Volume measured in U.S. gallons at a specified alcohol‑by‑volume level.
- **Liters Total**: Corresponding volume in liters.
- **Gallons Total**: Equivalent volume in U.S. gallons (rounded to one decimal place).
- **Cases**: Number of 12‑bottle cases that can be produced from the given proof gallon amount.

The table is useful for distillers and distributors when planning production, inventory, or shipping logistics.
