# What counts as a distillation cycle?

StillDragon explains that a *distillation* is defined by the TTB as one complete run through either a pot still or a column (reflux) still. Even if a plated column contains multiple plates, it still counts as **one** distillation because the spirit only completes a single full cycle of heating and cooling. The number of separation steps inside a column does not equate to multiple distillations for labeling purposes.

Key points:
- A pot still run = one distillation.
- Each plate in a reflux column is a *separation step*, but the whole column run remains one distillation.
- Packed columns are harder to quantify, yet they also count as one distillation per run.
- The TTB prohibits labeling spirits as "double" or "triple" distilled unless it reflects multiple full runs.
- Practical advantages of plated columns include higher reflux ratios, better separation, lower energy use, and reduced waste.

The article encourages distillers to consult legal counsel for compliance and offers StillDragon’s expertise in equipment selection.
