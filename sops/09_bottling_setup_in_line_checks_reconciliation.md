# SOP-09: Bottling Setup, In-Line Checks, and Reconciliation

## 1. Purpose
Control bottling startup, in-process quality checks, and material reconciliation to ensure compliant packaged product.

## 2. Scope
Applies to all bottling runs including changeovers and shutdown.

## 3. Roles
- Packaging Operator: setup and run line.
- Quality: in-line quality verification.
- Packaging Lead: line clearance and reconciliation approval.

## 4. Procedure
1. Pre-Run Setup
- Complete line clearance: remove previous product, labels, and coding materials.
- Verify correct bill of materials for SKU.
- Confirm filler settings, torque/cap settings, and code printer data.

2. Startup Checks
- Run initial bottles and inspect for fill level, closure integrity, label placement, and date/lot code legibility.
- Approve first-off sample before full-rate run.

3. In-Line Checks (Examples)
- Fill volume or net content checks at defined intervals.
- Closure torque or seal checks.
- Label position and barcode readability.
- Breakage, leakage, and foreign material observations.

4. Hold and Correction
- If check fails, stop line or isolate affected time window.
- Correct setup and verify restart with new first-off approval.

5. Reconciliation
- Reconcile spirit consumed versus bottles produced.
- Reconcile labels, closures, and cartons issued versus used versus returned.
- Investigate unexplained variance above threshold.

## 5. Small vs Large Operation Notes
- Small distillery: manual/semi-automatic lines require tighter operator frequency checks and second-person verification at key intervals.
- Larger operation: checkweighers, vision systems, and reject systems with electronic audit trails.

## 6. Records
- Line clearance checklist.
- In-line QC logs.
- Hold/rework documentation.
- End-of-run reconciliation report.
