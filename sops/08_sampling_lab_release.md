# SOP-08: Sampling and Lab Release

## 1. Purpose
Define representative sampling and release decisions for in-process and finished materials.

## 2. Scope
Applies to raw materials, wort/wash, distillate, maturation samples, and bottled product where applicable.

## 3. Roles
- Operator: obtain sample per plan and label correctly.
- Lab or Quality: perform analyses and release decisions.
- Production Lead: hold or release product based on results.

## 4. Procedure
1. Sampling Plan
- Define sample points, frequency, volume, and tests by material stage and risk.

2. Sample Collection
- Use clean, fit-for-purpose sample containers.
- Flush sample point as required.
- Label immediately with batch, vessel, date/time, sampler initials, and test request.

3. Chain of Custody
- Log sample transfer to lab.
- Maintain sample integrity (temperature/light controls where needed).

4. Testing
- Perform required tests (for example ABV, pH, gravity, sensory, microbiology, dissolved oxygen, package integrity).
- Review against approved specifications.

5. Release Decision
- Release when all criteria meet specification.
- If out of spec, place batch on hold and start non-conformance process.

## 5. Small vs Large Operation Notes
- Small distillery: may use external labs for some tests; define interim release rules clearly.
- Larger operation: in-house lab with LIMS, instrument calibration schedules, and trend analytics.

## 6. Records
- Sampling log.
- Lab test reports.
- Batch release or hold records.
- Non-conformance references.
