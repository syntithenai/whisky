# Distillery Study Tracker

Use this document alongside the main learning plan in README.md. The point is not to rush through brands. The point is to build a disciplined, comparable record of how distilleries explain themselves and how regional identities are constructed.

## How to Use This Tracker

For each distillery or brand:

1. Visit the official site.
2. Read the history or heritage page.
3. Read the process or craft page.
4. Identify the core range.
5. Note what the brand says about region, raw materials, stills, casks, and style.
6. Separate hard production facts from brand mythology.
7. Compare the official story with independent coverage later.

### Suggested Status Labels

- Not started
- Site reviewed
- History reviewed
- Process reviewed
- Core range mapped
- Compared with outside sources
- Notes complete

## Australia (Dedicated Section)

This section is intentionally large and designed as a living checklist. It includes established producers and smaller craft distilleries with active whisky releases or public whisky programs. Because Australian craft distilling changes quickly, treat this as a working register and update status notes as you verify operations, current bottlings, and ownership changes.

### Australia Study Method

1. Verify each official site and whether whisky is currently bottled, maturing, or archived.
2. Record whether the producer is primarily a whisky house or a multi-spirit distillery with a whisky line.
3. Track cask strategy, grain sourcing, climate effects, and local maturation style by state.
4. Mark distilleries that have closed, paused, or changed ownership.
5. Include a social-media check (especially Facebook) to find emerging producers before directory sites are updated.
